# 🧮 React Calculator

A modern and responsive calculator application built with React.js. This project performs basic arithmetic operations and features a clean, user-friendly interface.

## 🚀 Live Demo

https://azeenehsan09-afk.github.io/react-calculator/

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔄 Reset Calculator
- 📱 Responsive Design
- ⚡ Fast and lightweight

## 🛠️ Built With

- React.js
- JavaScript (ES6)
- CSS3
- Vite

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/azeenehsan09-afk/react-calculator.git
```

Navigate to the project:

```bash
cd react-calculator
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
